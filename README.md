# XDR与GPT部署方案向导

## 项目简介

这是一个用于指导XDR和GPT部署的Web应用程序。该工具提供了硬件兼容性检查、部署方案选择、部署步骤指导等功能，帮助技术人员快速完成XDR和GPT系统的部署工作。

## 功能特性

- **部署方案选择**：支持多种测试场景选择，包括XDR、DSP、网数融合、智能体等
- **硬件兼容性检测**：提供XDR和GPT硬件配置检查功能
- **部署拓扑说明**：展示XDR和GPT集群的整体部署拓扑
- **部署步骤指导**：提供详细的部署步骤和进度跟踪
- **虚拟化兼容性**：说明XDR和GPT在不同虚拟化平台上的兼容性
- **快速导航**：提供页面内快速跳转功能

## 技术栈

- **前端**：纯HTML5、CSS3和JavaScript
- **图标库**：Font Awesome 6.4.0
- **响应式设计**：支持桌面端和移动端访问

## 使用说明

### 部署方案选择

1. 在"部署方案选择"区域选择需要测试的场景
2. 点击"确定部署方案"按钮获取部署建议

### 硬件兼容性检测

1. 在"兼容性说明以及检测"区域选择XDR或GPT检测
2. 填写硬件配置信息
3. 点击"检测部署兼容性"按钮查看检测结果

### 查看部署详情

1. 点击"GPT部署"或"XDR部署"卡片
2. 使用顶部标签页切换不同详情内容：
   - 软件包及工具准备
   - 软件兼容性
   - 部署拓扑
   - 端口矩阵
   - 部署步骤

### 部署步骤跟踪

1. 在"部署步骤"标签页中，按照步骤进行操作
2. 完成每个步骤后点击"标记完成"按钮
3. 系统会自动更新部署进度
